树莓派安装 Tinc 记录

tinc 简介

简单地说，类似 zerotier
具体参加官网：tinc-vpn

从源码编译安装

sudo apt update
sudo apt install -y make libssl-dev zlib1g-dev liblzo2-dev libreadline-dev libncurses5-dev texinfo
curl http://www.tinc-vpn.org/packages/tinc-1.1pre11.tar.gz | tar xzvf -
cd tinc-1.1pre11
./configure
make -j4
sudo make install

从 apt 安装

sudo apt update
sudo apt install -y tinc
